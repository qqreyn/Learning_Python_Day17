# Learning_Python_Day17

Today's topic is virtual environments!

**Goal:** Create isolated Python environments for project dependency management

- **task1:** Create a new virtual environment from scratch

- **task2:** Activate the environment and verify that it is isolated

- **task3:** Install packages only inside the virtual environment

- **task4:** Deactivate and reactivate the environment to test persistence

- **task5:** Create a second virtual environment with different package versions

**mini-project:**
Project Environment Setup:

- Create a new project directory with its own virtual environment  
- Install specific packages inside the environment  
- Write a small script that uses those installed packages  
- Generate a `requirements.txt` file from the virtual environment  
- Document activation and setup instructions inside a `README.md` file
